Lessons from a Credit Card Fraud Detection Model

This project aimed to develop a machine learning model to detect fraudulent credit card transactions using an imbalanced dataset where fraud accounted for only 0.172% of all transactions. The dataset included PCA-transformed features for confidentiality. To address class imbalance, an undersampling approach was applied, creating a balanced subset of the data. Logistic regression was used for classification, achieving 90% accuracy on the test set. The project highlights the importance of handling imbalanced data and demonstrates how machine learning can be applied to enhance fraud detection in financial systems.

Diabetes Detection: A Machine Learning Approach

This project focused on predicting diabetes using the Pima Indians Diabetes Dataset. Logistic regression and SVM models were applied, achieving test accuracies of 77.9% and 77.5%, respectively. Logistic regression slightly outperformed SVM with a higher ROC AUC score of 0.8163, indicating good discriminatory ability. The dataset was slightly imbalanced, with 34.9% diabetic cases. Key features like Glucose, BMI, and Age likely contributed to the predictions. The project highlights the potential of machine learning in medical diagnostics, offering a reliable approach for diabetes prediction.

The Intersection of Wealth and Well-Being

This project explores the relationship between economic development and quality of life by analyzing data from 20 countries over a 10-year period (2010-2019). The key focus was to cluster countries based on their GDP and life expectancy, and build a classification model to predict which cluster a country belongs to. Through K-Means Clustering, countries were grouped into three clusters: wealthy nations (high GDP, high life expectancy), middle-income countries (medium GDP, moderate life expectancy), and developing nations (low GDP, low life expectancy). A Random Forest classifier was then trained, achieving 98% cross-validation accuracy and 100% test set accuracy in predicting cluster membership. The findings provide valuable insights for policymakers, offering recommendations on targeted investments for developing countries and strategies for sustainable growth in transitioning economies.
